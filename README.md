# 📊 Power BI Salary Dashboard + ETL (Power Query)

Projeto desenvolvido com foco em análise de dados e transformação ETL utilizando Power BI e Power Query.

## 📌 Objetivo
Criar um pipeline de transformação de dados de funcionários e construir um dashboard interativo com métricas salariais.

## 🖼️ Dashboard

## 📊 Indicadores Criados
- 💰 Salário Total
- 📉 Menor Salário
- 📈 Média Salarial
- 📊 Distribuição por Categoria (Alto, Médio, Baixo)
- 📌 Salário por Cargo

## 🛠️ Tecnologias Utilizadas
- Power BI
- Power Query (ETL)
- DAX

## 🔄 Processo ETL
Os dados passaram por:
- Limpeza de dados
- Remoção de duplicidades
- Padronização de texto
- Criação de colunas derivadas
- Classificação salarial
